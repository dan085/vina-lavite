# Instrucciones para Guardar las Imágenes

## Carpeta: `/public/images/`

Guarda las siguientes imágenes con estos nombres exactos:

1. **barriles.jpg** - Imagen de los barriles de roble con el logo "Viña La Vite"
2. **catacion.jpg** - Persona catando vino directamente del barril
3. **carmenere-vinedo.jpg** - Logo de Carmenere Mulchén 2009 sobre el viñedo
4. **evento.jpg** - Mesa elegante con decoración floral y vino La Vite

## Carpeta: `/public/images/vinos/`

Guarda las botellas de vino con estos nombres:

1. **malbec-colchagua.jpg** - Botella Malbec Gran Reserva D.O. Colchagua 2022 con montañas
2. **carmenere-aconcagua.jpg** - Botella Carmenere Gran Reserva D.O. Aconcagua 2022 con montañas
3. **cabernet-maipo.jpg** - Botella Cabernet Sauvignon Premium D.O. Maipo 2021 con montañas
4. **malbec-reserva.jpg** - Botella Malbec Gran Reserva vista frontal (cualquiera de las dos fotos frontales)

## Cómo guardar las imágenes:

### Desde el chat (si estás en VS Code):
1. Haz clic derecho en cada imagen
2. Selecciona "Save Image As..." o "Guardar imagen como..."
3. Navega a la carpeta correspondiente
4. Guárdala con el nombre exacto indicado arriba

### Desde Finder/Explorador:
1. Arrastra cada imagen desde el chat
2. Guárdala en el escritorio temporalmente
3. Renómbrala con el nombre correcto
4. Muévela a la carpeta correspondiente

## Verificación:

Después de guardar todas las imágenes, la estructura debe verse así:

```
vinalavite/
└── public/
    └── images/
        ├── barriles.jpg
        ├── catacion.jpg
        ├── carmenere-vinedo.jpg
        ├── evento.jpg
        └── vinos/
            ├── malbec-colchagua.jpg
            ├── carmenere-aconcagua.jpg
            ├── cabernet-maipo.jpg
            └── malbec-reserva.jpg
```

Una vez guardadas, recarga el navegador (http://localhost:4200) y las imágenes aparecerán automáticamente! 🍷✨
